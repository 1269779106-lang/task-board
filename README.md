# 任务看板 (Task Board)

一款精美的桌面任务管理工具，采用看板式设计，支持拖拽操作、分类管理、任务提醒。

界面参考 Microsoft To Do 和 Trello，使用 PySide6 构建现代化 UI。

## 功能特性

- **三列看板**：待办 → 进行中 → 已完成，拖拽流转任务状态
- **任务管理**：创建、编辑、删除、标记重要、设置优先级
- **分类管理**：自定义任务分类（工作/学习/生活等），支持增删
- **优先级标记**：高/中/低三级优先级，颜色直观区分
- **截止日期**：设置截止日期，过期任务自动标红提醒
- **任务提醒**：设置提醒时间，定时弹窗通知
- **数据持久化**：SQLite 本地存储，数据安全可靠
- **系统托盘**：最小化到系统托盘，不占任务栏

## 快捷操作

| 操作 | 方法 |
|------|------|
| 新建任务 | 点击顶部「+ 新建任务」按钮 |
| 编辑任务 | 双击任务卡片 |
| 删除任务 | 右键卡片 → 删除 |
| 移动任务 | 拖拽卡片到目标列 |
| 标记重要 | 右键卡片 → 设为重要 |
| 移动状态 | 右键卡片 → 移动到 |
| 分类筛选 | 点击筛选栏的分类按钮 |
| 管理分类 | 点击顶部「📁 分类」按钮 |

## 使用方法

### 方式一：直接运行 exe（推荐）

1. 下载 `TaskBoard.exe`
2. 双击运行，首次启动会自动创建数据库
3. 点击「+ 新建任务」开始使用

### 方式二：从源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/main.py
```

### 方式三：自行打包

```bash
# 安装打包工具
pip install pyinstaller

# 执行打包
python build.py

# 生成的 exe 位于 dist/TaskBoard.exe
```

## 数据存储

任务数据存储在用户 AppData 目录：

```
C:\Users\<用户名>\AppData\Roaming\TaskBoard\tasks.db
```

数据库包含两张表：
- `categories` — 分类表（名称、颜色、图标、排序）
- `tasks` — 任务表（标题、描述、状态、优先级、分类、日期、提醒等）

## 界面说明

### 顶部标题栏
- 显示应用名称和任务统计
- 「📁 分类」按钮：管理任务分类
- 「+ 新建任务」按钮：创建新任务

### 分类筛选栏
- 「全部」按钮：显示所有任务
- 分类按钮：点击筛选特定分类的任务

### 看板区域
- **待办列**：未开始的任务
- **进行中列**：正在执行的任务
- **已完成列**：完成的任务（卡片变灰、标题加删除线）

### 任务卡片
- 左侧色条：红色=紧急，橙色=普通，绿色=低优先级
- 右上角星标：标记为重要任务
- 底部：分类标签 + 截止日期（过期显示红色警告）

## 注意事项

### 首次运行

- 首次启动会自动创建数据库和默认分类（工作、学习、生活、其他）
- 可以通过「📁 分类」按钮添加、删除自定义分类

### 数据安全

- 数据存储在本地 SQLite 文件，不会上传到任何服务器
- 删除分类时，关联任务不会被删除，只是取消分类关联
- 建议定期备份 `AppData\Roaming\TaskBoard\tasks.db` 文件

### 拖拽操作

- 拖拽距离超过 10 像素才会触发（防止误触）
- 拖拽时显示卡片缩略图
- 拖拽到目标列后自动更新任务状态

### 提醒功能

- 提醒检查间隔为 30 秒
- 只有设置截止日期的任务才能设置提醒
- 已完成的任务不会触发提醒

### 界面交互

- 单击卡片：选中
- 双击卡片：打开编辑对话框
- 右键卡片：弹出操作菜单
- 拖拽卡片：移动到其他列

### 性能说明

- 空闲时内存占用约 50MB
- 数据库查询使用索引优化，响应迅速
- 卡片数量较多时（>100），滚动可能略有卡顿

## 常见问题

**Q: 任务数据保存在哪里？**
A: `C:\Users\你的用户名\AppData\Roaming\TaskBoard\tasks.db`

**Q: 如何备份数据？**
A: 复制 `tasks.db` 文件即可。恢复时将文件放回原位置。

**Q: 如何修改默认分类？**
A: 点击顶部「📁 分类」按钮，在弹窗中添加或删除分类。

**Q: 拖拽不生效怎么办？**
A: 确保拖拽距离超过 10 像素，且鼠标左键一直按住。

**Q: 提醒没有弹出？**
A: 提醒检查间隔为 30 秒，可能有短暂延迟。确保任务设置了截止日期和提醒时间。

**Q: 如何修改任务优先级？**
A: 双击任务卡片打开编辑对话框，修改优先级后保存。
